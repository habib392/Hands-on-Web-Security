Yeh room (jise TryHackMe par **"Search Skills"** ya **"Introductory Researching"** ke naam se jana jata hai) cyber security ki dunya ke aik bohot hi important aspect par focus karta hai: **Effective Researching & Information Gathering (OSINT)**.

As a penetration tester ya cyber security enthusiast, aapko har cheez pehle se maloom nahi ho sakti. Isliye yeh room aapko sikhata hai ke sahi cheez ko, sahi jagah par kaise dhoonda jata hai.
Is room ka aasan alfaz mein nishod (summary) yeh hai:

### 1. Information ko Evaluate Karna (Evaluation of Search Results)
Internet par har koi blog post ya article likh sakta hai, isliye har data par aankh band kar ke yakeen nahi kiya ja sakta. Yeh room aapko sikhata hai ke:

 * **Source** check karein (kya likhne wala koi reputable banda ya organization hai?).

 * **Evidence** check karein (kya unho ne solid facts diye hain?).

 * **Consistency** check karein (kya baqi reliable websites bhi wahi baat keh rahi hain?)*

### 2. Advanced Search Operators (Google Dorking)
Normal search ke elawa, cyber security mein targeted information nikalne ke liye specific operators use hote hain:

 * site:tryhackme.com — Sirf isi website ke results dikhayega.
 * filetype:pdf ya filetype:ppt — Kisi specific file type (jaise reports ya presentations) ko dhoondne ke liye.
 * "exact phrase" — Double quotes se wahi exact lafz search hota hai.
 * -tourism — Minus sign se woh specific word results se nikal jata hai.

### 3. Vulnerability aur Exploit Databases
Jab aapko penetration testing ke doran kisi service ka version maloom hota hai (e.g., Apache Tomcat 9.0.17), toh uske exploits dhoondne ke liye yeh resources kaam aate hain:

 * **NVD (National Vulnerability Database) & CVE Mitre:** Yeh public vulnerabilities ko **CVE IDs** (jaise CVE-2024-3094) ke sath track karte hain.
 * **Exploit-DB:** Yeh hackers aur red teamers ke liye sab se behtareen jagah hai jahan bane-banaye exploit codes (Proof of Concepts) milte hain.

 * **Searchsploit (Kali Linux):** Aapke Kali Linux mein Exploit-DB ka offline version pre-installed hota hai jise terminal par searchsploit [software_name] likh kar bina internet ke bhi use kiya ja sakta hai.

### 4. Specialized Search Engines & OSINT
 * **Shodan:** IoT devices, web servers, aur internet-connected hardware ko dhoondne ke liye use hota hai.
 * **Social Media (LinkedIn/Facebook):** Target company ke employees, unki technical skills, ya security questions ke answers (OSINT - Open Source Intelligence) gather karne ke liye use hota hai.

> **Nutshell mein:** Yeh room aapko aik "Search Expert" banata hai taake jab aap actual penetration testing ya labs solve kar rahe hon (jaise PortSwigger ya TryHackMe ke baqi rooms), toh aap atakne par sahi tarike se Google aur security databases ko leverage kar sakein.
